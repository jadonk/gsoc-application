I cross compiled on Arch Linux 64Bit, with following packages:
* arm-none-eabi-gcc (cross-compiler)
* qemu-user-static (AUR) (qemu runneer)

This are the commands I used:
    CC=arm-none-eabi-gcc make
    make run
